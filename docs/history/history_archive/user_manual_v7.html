Glass Box AI 診斷引擎 v7
使用手冊

本手冊說明 Glass Box AI 診斷引擎 v7 的完整操作流程，涵蓋蝕刻製程診斷工作站、技能管理，以及 AI Skill Builder Copilot 的使用方法。

系統需求與安裝

最低需求

項目需求
作業系統macOS 12+、Ubuntu 20.04+、Windows 10+
Python3.10 或更高版本
Node.js18.0 或更高版本
瀏覽器Chrome 100+、Firefox 100+、Safari 16+（需支援 Fetch API ReadableStream）
Anthropic API 金鑰必須具備有效的 ANTHROPIC_API_KEY

後端啟動

⚠️

重要：必須先 cd 進入 fastapi_backend_service/ 目錄後再執行 uvicorn，否則 Python 可能會載入 site-packages 中的同名模組（出現 NotImplementedError）。

cd fastapi_backend_service
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

前端啟動

cd fastapi_backend_service/frontend
npm install
npm run dev   # 預設啟動於 http://localhost:5173

圖 1-1：前後端啟動成功後，開啟瀏覽器即看到登入頁面

停止診斷與重置

停止進行中的診斷

診斷進行中，有兩種方式可以停止：

  • 點擊右側 Chat 面板頂部的 「停止」 按鈕（rose 色，僅在串流中顯示）
  • 點擊輸入框右側的 方形停止圖示（串流中切換為停止按鈕）

圖 10-1：串流進行中的停止按鈕（右上角 rose 色）

重置對話

點擊左側面板頂部的 「重置」 按鈕（RefleshCw 圖示），清除所有對話記錄、工具呼叫和報告，回到初始狀態。

⚠️

重置操作會清除目前所有診斷記錄，無法恢復。如需保留診斷報告，請先複製報告內容。

常見問題 · 19

常見問題（FAQ）

Q1：啟動後端顯示 NotImplementedError: coming soon

原因：您在錯誤的目錄執行 uvicorn，Python 載入了 site-packages 中的同名模組。

解決方法：確認您在 fastapi_backend_service/（含 main.py 的目錄）執行指令：

cd /path/to/fastapi_backend_service
uvicorn main:app --reload

Q2：診斷沒有反應，Chat 停在「Agent 執行中…」

可能原因：

  1. 後端未啟動（確認 http://localhost:8000/api/v1/health 回傳正常）
  2. ANTHROPIC_API_KEY 未設定或已失效
  3. 網路不穩定導致 SSE 中斷

解決方法：點擊「停止」按鈕，檢查後端 log，確認 API 金鑰有效後重試。

Q3：左側 Tab 沒有自動切換

可能原因：後端未正確發送對應 SSE 事件（tool_call、tool_result、report）。

解決方法：手動點擊 Tab 頁籤查看內容；確認後端實作正確發送這些事件。

Q4：Skill Builder 區塊 A 沒有顯示 AI 建議

可能原因：POST /api/v1/builder/suggest-logic 端點未實作或回傳錯誤。

解決方法：確認後端 builder API 已啟動，可用 curl 測試：

curl -X POST http://localhost:8000/api/v1/builder/suggest-logic \
  -H "Authorization: Bearer <your_token>" \
  -H "Content-Type: application/json" \
  -d '{"event_schema": {"event_type": "SPC_OOC_Etch_CD"}}'

Q5：Drawer 寬度在小螢幕上顯示異常

說明：Drawer 設定為 w-[60vw]，在 1024px 以下的螢幕上可能佔滿大部分畫面。

解決方法：建議在 1280px（Full HD）以上的螢幕使用，或修改 Drawer.jsx 和 SkillBuilderDrawer.jsx 的寬度設定。

Q6：登入後 Token 失效需要重新登入

說明：JWT Token 預設有效期為 24 小時（1440 分鐘）。

解決方法：重新登入即可。如需延長有效期，修改後端 ACCESS_TOKEN_EXPIRE_MINUTES 環境變數。
