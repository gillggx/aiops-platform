核心功能 #3

AI Skill Builder Copilot

製程工程師無需撰寫任何程式碼，透過三個步驟與 AI 協作，即可創建符合蝕刻製程知識的全新診斷 Skill。

圖 9：AI Skill Builder Copilot — 60vw 寬的三步驟建構介面
A

區塊 A：選擇事件類型 + 獲取 AI 診斷邏輯建議

從事件下拉選單選擇 SPC_OOC_Etch_CD，系統立即呼叫 POST /api/v1/builder/suggest-logic，AI 分析事件 Schema 後生成 PE（製程工程師）視角的診斷步驟建議，並以卡片形式展示（含「套用」按鈕）。

B

區塊 B：選擇 MCP 工具 + AI 自動欄位映射

以核取框勾選所需的診斷工具（mcp_check_recipe_offset、mcp_check_equipment_constants、mcp_check_apc_params）。每勾選一個工具，系統自動呼叫 POST /api/v1/builder/auto-map，AI 分析事件欄位與工具輸入 Schema 並生成映射表格，以信心度 badge（高/中/低）標示。

C

區塊 C：撰寫診斷邏輯 + AI 驗證

在 Textarea 中輸入診斷邏輯 Prompt（或直接套用 A 區塊的 AI 建議），點擊「驗證邏輯」呼叫 POST /api/v1/builder/validate-logic，AI 回傳驗證分數（0–100）、問題清單和改進建議。
